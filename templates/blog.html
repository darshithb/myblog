Introduction to Data Structures in Python!!

Posted on 3 Days Ago by darshithbadiyani

There are 4 built-in data structures in Python viz. list, tuple, dictionary and set.

First let’s know what a structure is?? A structure holds some related data together.

The life is made easy using these data structures.

Now, let’s dive into the topic and learn more about Python.

LIST- A list holds an ordered collection of items i.e. sequence of items in a list.
The elements of a List are enclosed within Square brackets [ ] and separated by a comma(,). Lists are mutable so we can make changes to the elements of the list on the go.

 

TUPLE- A tuple is used to hold together multiple objects. It is immutable in nature so we cannot change its values. They are enclosed within curved braces().

 

DICTIONARY- A dictionary is like an address book where you can find the address or contact details of a person by knowing only his/her name i.e. we associate keys(names) with values(details). Here the key must always be unique so that we can find the correct value for a key without any ambiguity.

A dictionary is placed within curly braces {}. Here, the keys must always be immutable, but the values can be either mutable/immutable.
A combination of the key and value are separated by a colon (:) and two consecutive pairs are separated by a comma.
Dictionaries are not necessarily ordered.

SET- A set is an unordered collection of simple objects. These are used when the existence of an object in a collection is more important than the order or how many times it occurs.
